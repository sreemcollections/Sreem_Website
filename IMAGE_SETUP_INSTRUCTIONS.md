# Panchaloham Jewelry Image Setup

Please save the panchaloham jewelry image (the collage with 7 jewelry pieces) as:
`src/assets/panchaloham-jewelry.png`

The image should be the one showing:
- Traditional temple jewelry necklaces
- Bangles/bracelets with green and red stones
- Earrings
- Artisan crafting jewelry in the center

Once you save this image to the assets folder, the Panchaloham Collection card will display it automatically.

The code has been updated to reference: `@/assets/panchaloham-jewelry.png`
