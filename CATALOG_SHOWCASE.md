# Catalog Showcase Mode

This website is configured as a **catalog showcase only**. Users can browse and view products but cannot make purchases.

## Features

### Product Display
- Product cards with hover zoom animation
- Image slider on hover (4 images per product)
- Smooth transitions and luxury animations
- "Catalog Item" badge on all products

### Product Details
- Full product information and specifications
- Pricing display for reference
- GST information
- Artisan and region details
- No purchase buttons (Add to Cart, Buy Now removed)
- "Request Information" button for inquiries

### User Experience
- Hover to zoom into product images
- Cycle through product images with interactive dots
- "View Details" button on product cards
- Save to Wishlist functionality
- Share product capability
- Mobile-optimized viewing experience

## Removed Features
- Shopping cart
- Checkout process
- Add to Cart buttons
- Buy Now buttons
- Payment integration

## Purpose
The catalog mode allows customers to:
1. Browse the complete product collection
2. View detailed product information
3. Save favorite items to wishlist
4. Share products with others
5. Request information about specific items
6. Explore craftsmanship and heritage

This is ideal for luxury brands that prefer to handle sales through personal consultation or custom orders.
